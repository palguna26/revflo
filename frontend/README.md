# RevFlo Frontend

This is the frontend application for **RevFlo** - The autonomous AI code review platform.

## Project Structure
- Built with **React** + **Vite** + **TypeScript**
- UI Components: **Shadcn UI** + **Tailwind CSS**
- State Management: React Hooks
- Routing: React Router DOM

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Production Build

```bash
# Build for production
npm run build
```

## Key Features
- **Dashboard**: Central hub for repo health and activity.
- **AI Feed**: Real-time insights and recommendations.
- **Repo Analysis**: Deep dive into PRs and Issues with AI validation.
